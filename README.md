# ✋ Hand Gesture Recognition using Machine Learning

## 📌 Overview
This project builds a Hand Gesture Recognition system using Machine Learning.  
It classifies different hand gestures from images using HOG features and SVM.

---

## 🚀 Features
- Image preprocessing (grayscale + resize)
- Feature extraction using HOG
- Classification using SVM
- High accuracy performance
- Model saving for reuse

---

## 📂 Dataset
Dataset used: LeapGestRecog

---

## ⚙️ Tech Stack
- Python
- NumPy
- PIL
- Scikit-learn
- Scikit-image

---

## 🧠 Workflow
1. Load dataset
2. Preprocess images
3. Extract HOG features
4. Encode labels
5. Train SVM model
6. Evaluate performance
7. Save trained model

---

## 📊 Results
- Achieved high accuracy on test data
- Robust classification across multiple gestures
